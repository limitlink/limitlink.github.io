.htaccessを使ったアクセス制限

.htaccessを使ってドメインやIPアドレスでアクセスを制限することができます。

下記のコードを記述したファイルを「.htaccess」というファイル名でサーバーにアップロードしてください。
.htaccessを設置したディレクトリ以下のディレクトリにアクセスを制限できます。

IPアドレスが「192.168.1.1」からのみアクセスを許可したい場合。 
Order deny,allow
deny from all
allow from 192.168.1.1
IPアドレスが「192.168.1.」から始まるIPアドレスのみアクセスを許可したい場合。 
Order deny,allow
deny from all
allow from 192.168.1.
ホストが「.example.com」からのみアクセスを許可したい場合。 
Order deny,allow
deny from all
allow from .example.com

最後の行には改行を入れてください。
ファイル転送時はテキストモードかアスキーモードで転送してください。
「.(ドット)」から始まるファイル名を付けられない場合は、 htaccess.txt等のファイル名で保存し、サーバーにアップロード後 「.htaccess」に変更してください。
